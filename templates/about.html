About This Project

This web app demonstrates sentiment analysis on movie reviews using a machine learning pipeline built with TF-IDF, Logistic Regression, and LIME explainability.

  • Preprocessing: Cleans and vectorizes text for model input.
  • Model: Logistic Regression on TF-IDF features, trained to identify positive and negative sentiments.
  • Interpretability: LIME (Local Interpretable Model-Agnostic Explanations) highlights which words influenced each prediction.
  • Interface: Flask and Bootstrap provide an easy-to-use web dashboard with prediction, batch upload, word clouds, and performance visualizations.

Created as a final year MCA project to showcase modern AI interpretability on real-world data, with transparent, user-friendly results.
